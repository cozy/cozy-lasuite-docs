# cozy-lasuite-docs

## What is it

cozy-lasuite-docs is a container app that embeds Docs app. It is a very basic app displaying the cozy-bar and an iframe. By leveraging [cozy-external-bridge](https://github.com/cozy/cozy-libs/tree/master/packages/cozy-external-bridge), it allows history syncing from Docs and exposes APIs to Docs.

## Install

You can then clone the app repository and install dependencies:

```sh
$ git clone https://github.com/cozy/cozy-lasuite-docs.git
$ cd cozy-lasuite-docs
$ yarn install
```
